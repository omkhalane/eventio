# Asset Strategy

No generated image assets are required for this cleanup pass. Asset direction is
captured as text descriptors so designers and maintainers can create final
brand assets deliberately.

Descriptor files live in `apps/web/public/assets/descriptors`.

Required descriptors:

- `logo.txt`
- `favicon.txt`
- `banner.txt`
- `og-image.txt`

Production assets should be optimized, accessible, and versioned like source
files. Avoid committing large exports or exploratory design files.
