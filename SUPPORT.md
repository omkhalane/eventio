# Support

Eventio is an open-source project maintained by Om Khalane.

## Where to Ask

- Bugs: open a bug report using the issue template.
- Feature ideas: open a feature request.
- New sources: use the new scraper source template.
- Security issues: follow [SECURITY.md](SECURITY.md).

## Before Opening an Issue

- Search existing issues.
- Run the relevant local checks if possible.
- Include environment details and reproduction steps.
- Remove secrets, tokens, API keys, and private user data from logs.
